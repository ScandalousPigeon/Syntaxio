# Syntaxio: CAB302 Project

Group Members: Leon, Mimi, Xihan

Syntaxio is a beginner-friendly coding application designed to help new programmers learn through guided problem solving rather than direct answers.

The goal of Syntaxio is to support users as they write, test, debug, and improve their own code. Users can attempt programming challenges, run their solutions, view test results, track their progress, and request guidance when they get stuck.

Unlike applications that simply provide complete solutions, Syntaxio focuses on helping users develop their own problem-solving skills.

---

## Project Overview

Syntaxio is designed as a coding learning platform for beginner programmers.

Users select a programming challenge, write their own solution in the built-in editor, run their code, and receive feedback based on automated test results. When users need help, the application can provide hints, pseudocode guidance, and learning resources without giving away the full solution.

The application is designed to work fully without AI. AI is used only as an optional enhancement to improve the learning experience.

---

## Core Features

### Non-AI Features

Syntaxio includes the following core features:

- Programming challenges for users to solve
- Built-in code editor
- Solution runner to execute user code
- Automated checking system to evaluate submitted solutions
- Test results display, including:
  - Pass/Fail results
  - Expected output
  - Actual output
- Challenge browser
- Sorting challenges by difficulty
- Solution history
- Hint log
- Progress tracking over time

These features allow Syntaxio to operate as a complete learning platform even without AI integration.

---

## AI Features

AI is used as an additional learning support tool. It is not required for the core functionality of the application.

The AI can:

- Provide context-aware hints
- Generate pseudocode guidance
- Suggest helpful learning resources
- Give feedback on a working solution
- Suggest alternative approaches or improvements

The AI is designed to avoid providing full complete solutions. Its purpose is to guide users toward the answer, hereby encouraging independent thinking.

---

## User Flow

1. User selects a coding challenge
2. User writes code in the editor
3. User runs their solution
4. The system displays test results
5. User can request a hint or feedback
6. User improves their solution and tries again

---


## Design Goal

The main goal of Syntaxio is to create a supportive environment where beginner programmers can practice coding, learn from mistakes, and build confidence.

Rather than giving users complete answers, Syntaxio encourages them to think through problems, improve their solutions, and develop stronger programming skills over time.
